Serve & Build

Documentation of development server and build process

Installation

Done uses Gulp for development serve and build process

  • 1. Make sure NodeJs installed on your computer.
  • 2. Open terminal and navigate to root /done-design-system directory and run npm install to install dependencies.

Development server

Run npm start to start local development server. Open http://localhost:3000/ in browser.

Build

Run npm run build to build your project. All files including compiled & minified css, js files will be stored in /dist directory.
